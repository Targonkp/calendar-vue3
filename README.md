# Компонент календаря на Vue3

1) **Использован Vue3 (Composition Api) + TS + VCalendar**
2) **Реализована смена языка - для этого нужно передавать язык в locale**
3) **При инициализации компонент может принимать свойство даты, например: '2025-11-01'**
4) **При клике по любому дню можно получить его дату**
5) **Запуск проекта: https://targonkp.github.io/calendar-vue3/**
